# Bachelor thesis

- Export thesis report:
	```bash
	pnpm run thesis-build
	```
- Show thesis slide:
	```bash
	pnpm run slide-run
	```
- export thesis slide:
	```bash
	pnpm run slide-export
	```
	- With animation:
		```bash
		pnpm run slide-export --with-clicks
		```
	- Dark theme:
		```bash
		pnpm run slide-export --dark
		```
